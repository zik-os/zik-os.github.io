This is converter Opera notes to Vivaldi notes


  1. Обновите Opera 12.17 до Opera Blink.
  2. После обновления на Рабочем Столе появится "Opera 12 Notes.html".
  3. Скопируйте содержимое этого файла сюда и нажмите "Convert".
  4. Получившийся JSON скопируйте в текстовой файл заметок Vivaldi.
    (Обычно он находится по адресу %AppData%\Local\Vivaldi\User Data\Default\Notes)
  5. Перезагрузите Vivaldi.
